### 探知术 Scrying

*5环预言*

**施法时间：** 10分钟

**施法距离：** 自身

**法术成分：** V、S、M（一个至少价值1000gp的法器，例如水晶球、银镜或盛有圣水的池盆）

**持续时间：** 专注，至多10分钟

你可看见并听见一个由你选择、且与你处于同一位面的特定生物。目标必须进行一次感知豁免，豁免结果会因你对目标熟悉程度及你与目标的实体联系而修正。若目标知道你正在施展此法术，且愿意被观测，它可自愿令该豁免失败。

**表：探知豁免修正**

| 认知程度 | 豁免修正 |
|----------|----------|
| 间接认知（你仅听说过目标） | +5 |
| 直接见过（你见过目标） | +0 |
| 熟识（你很了解目标） | -5 |
| 实体联系 | 豁免修正 |
| 画像或肖像 | -2 |
| 私人物品或衣物 | -4 |
| 身体组织、发丝、指甲碎屑等 | -10 |

豁免成功则目标不受影响，且你在24小时内不能再次对其施展该法术。

豁免失败则法术会在目标10尺内创造一个不可见的感应器。你可如同置身其处一般通过该感应器视听。该感应器会随目标移动，并在持续时间内保持在其10尺内。能看见隐形物体的生物会将该感应器视作一个发光球体，大小约如你的拳头。

你也可以不以生物为目标，而改选一个你见过的地点作为目标。如此施法时，感应器会出现在该地点且不会移动。
