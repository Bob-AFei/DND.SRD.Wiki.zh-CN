### 操控天气 Control Weather

*8环变化*

**施法时间：** 10分钟

**施法距离：** 自身（半径5里）

**法术成分：** V、S、M（燃烧的熏香，以及与水混合的泥土和木屑）

**持续时间：** 专注，至多8小时

在持续时间内，你可掌控自身周围半径5里内的天气。你必须在户外施放该法术。若你移动到无法直通天空的地点，法术会提前结束。

施法时，你改变当前天气状态。当前状态由GM依据当地气候与季节决定。你可以改变降水、温度和风力。新天气要在1d4 × 10分钟后生效。生效后你可以再次改变天气。法术结束时，天气会逐步恢复正常。

当你改变天气时，在下表中找到当前状态，并将其阶段上调或下调一级。改变风力时，你还可以改变风向。

**表：操控天气（降水）**

| 阶段 | 状态 |
|------|------|
| 1 | 晴朗 |
| 2 | 薄云 |
| 3 | 阴天或地雾 |
| 4 | 雨、冰雹或雪 |
| 5 | 暴雨、强雹或暴风雪 |

**表：操控天气（温度）**

| 阶段 | 状态 |
|------|------|
| 1 | 难以忍受的炎热 |
| 2 | 炎热 |
| 3 | 温暖 |
| 4 | 凉爽 |
| 5 | 寒冷 |
| 6 | 极寒 |

**表：操控天气（风力）**

| 阶段 | 状态 |
|------|------|
| 1 | 无风 |
| 2 | 中等风力 |
| 3 | 强风 |
| 4 | 烈风 |
| 5 | 风暴 |
